# Outline

## Core question
When existing observations can no longer eliminate competing worlds, why does action become part of reasoning itself?

## Main thread

1. Several worlds can remain alive at the same time.
   - Multiple explanations may still fit all existing observations.
   - This is not necessarily a failure of reasoning; it may be a limit of available information.

2. Thinking harder does not always help.
   - Reprocessing the same evidence does not automatically create new constraints.
   - At some point, reasoning stalls because the information has been exhausted.

3. Action can be used to generate distinguishing observations.
   - In Werewolf: applying pressure, forcing a PK, balancing, voting, withholding a vote, or changing sides.
   - The purpose is not always to reach an immediate answer, but to make different worlds produce different reactions.

4. These actions are forms of experiment design.
   - A useful action is one that separates competing explanations.
   - The same structure appears in scientific experiments, diagnosis, product testing, and exploration in reinforcement learning.

5. New observations create new constraints.
   - Reactions, votes, rescues, silence, and changes of position become fresh evidence.
   - These observations allow further pruning.

6. Reasoning is a loop, not a line.

   Observation → Possible Worlds → Pruning → Action → New Observation

7. The goal is not necessarily to recover a unique truth before acting.
   - The practical goal is to improve decisions under uncertainty.
   - Action is not the end of reasoning; it is how reasoning continues.

## Ending bridge
Information is not always passively given. Through action and interaction, new information can be generated.
